## How to run

To run this project, download folder with code from this repository and unzip it.

```bash
  double click on index.html
```

## How to play

#### Controls

player 1 (left)

| Key     | Description |
| :------ | :---------- |
| `a`     | Move left   |
| `d`     | Move right  |
| `w`     | Jump        |
| `Space` | Hit         |

player 2 (right)

| Key          | Description |
| :----------- | :---------- |
| `ArrowLeft`  | Move left   |
| `ArrowRight` | Move right  |
| `ArrowUp`    | Jump        |
| `ArrowDown`  | Hit         |

#### Reset and start over

Restart your browser.
